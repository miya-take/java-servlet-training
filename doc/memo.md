# 01 HTMLとWebページ
- タグ
  - `<tag>contentc</tag>`
  - 開始タグと終了タグのペアで囲む
- 空要素
  - HTMLタグの中には空要素と呼ばれる内容を持たないタグがある。
- 属性
  - タグに加える補足的な情報
  - `<tag 属性名="値">---</tag>`
  - 複数の属性で区切る場合には、半角スペースで区切る
- HTMLの基本構造

```
<!DOCKTYPE> // HTMLのバージョンを指定

<html> //HTML全体をなdのタグで囲む。
  // タイトルもiphone コード、昨夜などのWebページに関する情報を記述する
  // ただし、タイトル以外の情報はブラウザに表示されない
  <head>
    ...
  </head>
  
  // タグの内容に、ブラウっざに表示されるWebページ本体を
  <body>
    ...
  </body>
</html>
```
